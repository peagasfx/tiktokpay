🤖 Configurar Notificações Telegram

Configuração automática em 3 passos simples

1 Criar Bot no Telegram
  1. Abre o Telegram no teu celular ou computador
  2. Procura por @BotFather e inicia conversa
  3. Envia o comando: /newbot
  4. Dá um nome ao bot (ex: "Notificações Funil")
  5. Dá um username (deve terminar em "bot", ex: "meufunil_notificacoes_bot")
  6. Guarda o TOKEN que o BotFather te dá
📱 Abrir @BotFather no Telegram
2 Obter Chat ID
  1. Procura pelo teu bot no Telegram (pelo username que criaste)
  2. Inicia uma conversa com ele
  3. Envia qualquer mensagem (ex: "Olá")
  4. Cola o TOKEN abaixo e clica em "Obter Chat ID"
3 Configurar e Testar

Cola o TOKEN e Chat ID abaixo. O sistema vai configurar automaticamente e testar a conexão.
